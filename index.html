🌸 One-time Setup

To make photos visible on every device, connect a free cloud database.
This only takes 2 minutes — you'll never need to do it again.

  • 1 Go to console.firebase.google.com → Create project (any name, disable Analytics) → Continue
  • 2 Click Realtime Database (left menu) → Create Database → choose any location → Start in test mode → Enable
  • 3 Copy the database URL shown (looks like https://your-project-default-rtdb.firebaseio.com) and paste it below
